# PnBook - Backend (pnbook-be)

The backend project for the PnBook system, is built with Expressjs and TypeScript.

## Tech Stack

- **Core:** Expressjs ^5.2.1, TypeScript
- **Code Quality:** ESLint (Flat Config), Prettier, Husky, Lint-staged, Commitlint

## Setup requirement

- Node.js >= 18
- Package Manager: npm/pnpm/yarn

## Install & run the project

1. **Clone project:**
   ```
   git clone https://github.com/TaiVo1905/pnbook-be-repo.git
   cd pnbook-be-repo
   ```
2. **Install dependencies**

   ```
   npm install
   ```

3. **Start project**
   ```
   npm run dev
   ```

## Branch name and commit discription

- **Branch name:** type/JIRA-KEY-discription (Example: feat/PNBOOK-1-login)
- **Commit discription:** type(JIRA-KEY): discription (Example: feat(PNBOOK-1): add button component)
- **Type:** `feat`, `fix`, `chore`, `style`, `refactor`, `test`...
- **JIRA-KEY example:** `PNBOOK-1`
